# 素材库与方案拼装 — 用户指南

## 概述

PPTagent 支持从历史 PPT 中提取可复用素材，与 AI 生成内容混合拼装，输出风格一致的方案 PPT。每张页保留来源标注，便于审计溯源。

---

## 1. 素材库管理

### 1.1 上传样本

1. 进入「知识库」页面
2. 拖拽 PPTX/PDF/DOCX 文件到上传区域（单文件 ≤ 50MB，单次 ≤ 20 份）
3. 系统自动解析并提取每页素材

### 1.2 搜索素材

进入「素材库」页面，支持以下搜索方式：

- **关键词搜索**：输入标题或正文关键词
- **视觉类型筛选**：封面 / 目录 / 架构图 / 流程图 / 数据图 / 正文 / 结尾 / 混合
- **行业标签筛选**：按行业分类过滤

### 1.3 素材详情

点击素材卡片打开详情面板，可查看：

- SVG 预览 / 缩略图大图
- 标题、正文、页码
- 色板、字体、行业标签
- 来源样本信息

## 2. 方案拼装

### 2.1 创建草稿

1. 进入「草稿」页面
2. 点击「新建草稿」，输入方案标题
3. 进入草稿编辑器

### 2.2 添加页面

支持三种来源：

| 来源类型 | 说明 | 标签颜色 |
|---------|------|---------|
| **复用** | 从素材库插入的历史页 | 蓝色 |
| **AI 生成** | 由 Agent 根据轨迹生成 | 紫色 |
| **手动** | 手动创建的空白页 | 灰色 |

**插入素材页**：
1. 在素材库中找到目标素材
2. 点击「插入草稿」按钮
3. 系统自动执行风格归一（色板/字体对齐）

### 2.3 编辑草稿

- **拖拽排序**：长按页面卡片拖动调整顺序
- **删除页面**：点击页面卡片上的删除按钮
- **自动保存**：每 5 秒自动保存，支持乐观锁并发控制

### 2.4 草稿锁定

- 打开草稿时自动获取编辑锁（30 分钟有效期）
- 其他用户打开同一草稿时显示为只读模式
- 锁到期后自动释放，也可手动释放

---

## 3. 导出 PPTX

### 3.1 发起导出

1. 在草稿编辑器中点击「导出 PPTX」
2. 系统创建导出任务并返回任务 ID
3. 实时显示导出进度

### 3.2 来源标注

导出的 PPTX 中每张页包含来源元数据：

- `pptagent:sourceType`：reused / generated / manual
- `pptagent:sourceAttribution`：来源描述
- 复用页附加 `sample_id` 和 `page_index`

### 3.3 下载

导出完成后可直接下载 PPTX 文件。

---

## 4. 数据安全

### 4.1 PII 检测

上传样本时自动检测以下 PII 类型：

- 手机号码
- 电子邮箱
- 身份证号
- 银行卡号
- 客户姓名
- 地址

PII 字段在摘要中被标注，可选择替换或保留。

### 4.2 数据导出

在「安全」页面可一键导出所有数据（ZIP 格式），包含：

- 原始样本文件
- 解析结果
- 偏好设置 JSON
- SHA-256 校验清单

### 4.3 数据删除

一键删除所有数据：

- 生产库数据：24 小时内清除
- 备份数据：7 天内清除

## 5. 常见问题

**Q: 素材搜索很慢怎么办？**
A: 确保 PostgreSQL 已启用 pgvector 扩展并创建了 HNSW 索引。首次搜索可能较慢（冷启动），后续查询会显著加速。

**Q: 风格归一失败会影响导出吗？**
A: 不会。归一失败时系统会保留原始样式并标记 `normalized_failed=true`，导出时仍会包含该页面。

**Q: 草稿锁过期了怎么办？**
A: 系统每 5 分钟自动清理过期锁。您也可以手动释放锁后重新获取。

**Q: 如何查看页面来源？**
A: 在草稿编辑器中，每张页面卡片显示来源标签（复用/AI/手动）。导出的 PPTX 中也包含来源元数据。
